## 21.How to empty an array?

```js
let arr = [1, 2, 3, 4, 5];
```

```js
arr.length = 0;
```
```js
arr.splice(0, arr.length);
```

## 22.How can you Replace an element at a specific index in an Array?


```js
let a = [1 , 2 , 3 , 4]
```

```js
a[2] = 100;

console.log(a)
```

## 23.How can you Delete an element at a specific index in an Array?

```js
let a = [1 , 2 , 3 , 4]
```

```js
a.splice(2,1);

console.log(a)
```
